OWASP Broken Web Applications

This is the VM for the Open Web Application Security Project (OWASP) Broken Web Applications project. It contains many, very vulnerable web applications. More information about this project can be found at http://code.google.com/p/owaspbwa/.

!!! This VM has many serious security issues, we strongly recommend that you run
it only in "host only" or "NAT" network in the virtual machine settings !!!

This VM has two web servers running. One Apache server on port 80 and one Tomcat server on port 8080. The following vulnerable web applications are running on the VM (listed in no particular order).
